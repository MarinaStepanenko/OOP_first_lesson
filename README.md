# E-commerce Core System

Проект представляет собой ядро для интернет-магазина, реализующее основные сущности электронной коммерции. Система подготавливает фундамент для дальнейшей разработки различных интерфейсов (веб-сайт, телеграм-бот и т.д.).

## Архитектура проекта

### Структура директорий
````
project_root/
├── src/ # Исходный код приложения
│ ├── init.py
│ ├── product.py # Модуль класса Product
│ └── category.py # Модуль класса Category
├── tests/ # Тесты
│ ├── init.py
│ ├── test_product.py # Тесты для класса Product
│ └── test_category.py # Тесты для класса Category
└── README.md # Документация
````

## Описание классов

### Класс `Product`

Класс представляет товар в интернет-магазине.

**Атрибуты:**
- `name` (str) - название товара
- `description` (str) - описание товара
- `price` (float) - цена товара
- `quantity` (int) - количество товара в наличии

**Пример использования:**
```python
product = Product(
    name="LG PC",
    description="Это описание продукта",
    price=156.78,
    quantity=2
)
```
# Класс `Category`

## Описание
Класс `Category` представляет собой категорию товаров в интернет-магазине. Каждая категория содержит список товаров определенного типа и автоматически участвует в подсчете общей статистики магазина через статические атрибуты класса.

## Структура класса

### Атрибуты экземпляра
| Атрибут | Тип данных | Описание | Обязательность |
|---------|------------|----------|----------------|
| `name` | `str` | Название категории | Обязательный |
| `description` | `str` | Описание категории | Обязательный |
| `products` | `list[Product]` | Список товаров, принадлежащих категории | Обязательный |

### Атрибуты класса (статические)
| Атрибут | Тип данных | Описание | Поведение при инициализации |
|---------|------------|----------|----------------------------|
| `category_count` | `int` | Общее количество созданных категорий | Увеличивается на 1 при создании новой категории |
| `product_count` | `int` | Общее количество всех товаров во всех категориях | Увеличивается на сумму количеств (`quantity`) всех товаров в новой категории |

## Параметры конструктора

### Обязательные параметры

#### 1. `name`
**Тип:** `str`  
**Описание:** Название категории товаров  
**Примеры:**
- `"Смартфоны"`
- `"Ноутбуки"`
- `"Бытовая техника"`

**Требования:**
- Не может быть пустой строкой
- Должен быть читаемым и информативным
- Рекомендуется использовать заглавную букву в начале

#### 2. `description`
**Тип:** `str`  
**Описание:** Подробное описание категории  
**Примеры:**
- `"Мобильные телефоны различных брендов и ценовых категорий"`
- `"Портативные компьютеры для работы и игр"`

**Требования:**
- Может быть пустой строкой (но не рекомендуется)
- Должен описывать общие характеристики товаров категории
- Максимальная длина не ограничена

#### 3. `products`
**Тип:** `list[Product]`  
**Описание:** Список товаров, принадлежащих категории  

**Структура списка:**
```python
[
    Product(name="...", description="...", price=100.0, quantity=5),
    Product(name="...", description="...", price=200.0, quantity=3),
    # ... другие товары
]
```
